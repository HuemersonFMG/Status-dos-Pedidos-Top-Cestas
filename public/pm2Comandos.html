>>>LISTA GERAL DE COMANDOS:

Comando O que faz
pm2 start server.js --name StatusPedidos Inicia a aplicação Node.js com nome definido.
pm2 list Lista todos os processos gerenciados pelo PM2.
pm2 status Mostra status, CPU, memória, reinícios e modo de execução.
pm2 logs StatusPedidos Exibe logs em tempo real da aplicação.
pm2 logs StatusPedidos --lines 100 Mostra as últimas 100 linhas de log.
pm2 restart StatusPedidos Reinicia a aplicação.
pm2 restart StatusPedidos --update-env Reinicia aplicando variáveis de ambiente atualizadas.
pm2 reload StatusPedidos Recarrega a aplicação com menor interrupção possível.
pm2 stop StatusPedidos Para a aplicação, mas mantém registrada no PM2.
pm2 delete StatusPedidos Remove a aplicação da lista do PM2.
pm2 show StatusPedidos Mostra detalhes completos do processo.
pm2 describe StatusPedidos Alternativa ao show, exibe configurações e caminhos de log.
pm2 monit Abre painel de monitoramento com CPU, memória e logs.
pm2 flush Limpa os logs de todos os processos.
pm2 save Salva a lista atual de processos para restaurar após reboot.
pm2 resurrect Restaura processos salvos anteriormente com pm2 save.
pm2 startup Configura o PM2 para iniciar junto com o Windows/servidor.
pm2 kill Encerra o daemon do PM2 e todos os processos gerenciados.
pm2 ping Verifica se o daemon do PM2 está ativo.
pm2 env 0 Mostra variáveis de ambiente do processo pelo ID.
pm2 reset StatusPedidos Zera contador de reinicializações do processo.
pm2 restart all Reinicia todos os processos PM2.
pm2 stop all Para todos os processos PM2.
pm2 delete all Remove todos os processos do PM2.
